# BareBlocks — brand kit

The identity is a **light technical workspace**: a bright white canvas with a violet
brand accent and mono "blueprint annotation" callouts. Two-color system —
**violet = brand/UI**, **blue = the product** (the add-on's grid is blue, so renders
keep their blue and live inside dark "viewport" panels that pop against the white page).

## Name & tagline
- **Name:** BareBlocks (one word, camel B's)
- **Primary tagline:** Block out worlds, not geometry.
- **Descriptor:** Parametric blockout & greyboxing kit for Blender.

## Logo
- Isometric cube of three blue faces (`assets/favicon.svg`). Top `#6aa6ff`, left
  `#2f74e0`, right `#1c4f9e`. Pairs with the wordmark set in Space Grotesk 700.
- The logo stays **blue** on purpose — it's the product (the blocks). It sits happily on
  the white page next to violet UI. Clear space ≥ the cube's width; never recolor off-palette.

## Color
| Token | Hex | Use |
|-------|-----|-----|
| bg | `#ffffff` | page background |
| bg2 | `#f6f3fc` | alternating section bands (stats, video) |
| surface | `#ffffff` | cards |
| panel | `#120c1d` | dark "viewport" frames behind product renders |
| line | `#e8e3f3` | hairlines, borders |
| purple | `#7c3aed` | brand primary — buttons, links, accents |
| purple-deep | `#5b21b6` | eyebrows, prices, code text |
| lilac | `#a78bfa` | tick labels on dark panels |
| ink | `#1b1430` | headings, body text |
| muted | `#6c6483` | secondary text |

Violet is the brand; blue belongs to the product renders. Keep violet purposeful —
buttons, one link color, the play button, step numbers. The dark panels carry the blue.

## Type
- **Display:** Space Grotesk (600–700) — headings, wordmark, stat numbers.
- **Body:** Inter (400–600).
- **Utility / mono:** JetBrains Mono — eyebrows, dimension callouts, labels, code.
  The mono "blueprint annotation" (e.g. `GRID 1.0 m`, `x 26.0 · y 26.0`) is the
  signature device; use it for anything technical or measured.

## Voice
- Plain, specific, active. Name things by what the user controls.
- Lead with what it does, not adjectives. "Bend the path" beats "powerful spline system."
- Sentence case everywhere except mono labels (which are UPPERCASE).

## Signature element
The **dark "viewport" panels** holding the blue renders against the white page (with violet
corner brackets and mono dimension ticks) — the product looks like it's lit on a workbench.
Don't add competing decoration around them.
